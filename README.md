# OnBoarding

## Requirements

```plain
dotnet 8.0
c# 12.0
```

## Clone

```sh
git clone https://github.com/s2quake/OnBoarding.git --recursive
```

## Build and Run

```sh
dotnet build
dotnet run --project OnBoarding.ConsoleHost
```

## Test

실행후 프롬프트에서 다음 명령어를 입력합니다.

```plain
action add 3
```

> ``exit`` 명령어 입력시 프로그램이 종료됩니다.
